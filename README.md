# Sotto

A quiet gallery for AI music creators. Invitation-only.  
AI音楽クリエイターのための、招待制の小さなギャラリー。

## Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS v4
- Supabase (Auth / DB / Storage)
- Vercel (Hosting)

## 開発

```bash
npm install
npm run dev
```

http://localhost:3000

## 環境変数

`.env.local` に以下を設定:

```
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=
```
